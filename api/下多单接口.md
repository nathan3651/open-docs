
## 下多单接口
POST
`
https://rfinex.com/api/v1/orders/multi
`

参数在query部分传递

```
  access_key: 从平台获取的access_key
  market: 市场
  signature: 签名
  tonce: 时间戳，精确到秒
  orders: 多单数组的json字符串
```

```json
[
	{
  		price: 价格
	  	volume: 数量
  		side: "buy"
	},
	{
	  	price: 价格
  		volume: 数量
  		side: "sell"
	}
]
```
